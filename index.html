📖 如何使用本工具

寫粵語歌最大的挑戰在於「啱音」（詞的聲調必須與旋律高低一致）。本工具內建豐富的基督教詞庫，幫助你快速配對填詞。

🎼 聲調查字 (Tone Search)

當你已經寫好了一段旋律，需要找對應聲調的詞語填進去時使用：

  1. 聽出旋律的高低音走向。
  2. 點擊畫面上方的數字按鈕加入聲調（1=高音，4=低音 等）。
  3. 例如：你想找一個先「低音」後「高音」的詞，就依序點擊 4 和 1。
  4. 點擊「查詞語」，系統會幫你找出如「耶穌」、「平安」等符合聲調的詞。

🏷️ 主題詞庫 (Topical Dictionary)

當你缺乏靈感，需要尋找特定主題的用詞時使用：

  1. 點擊畫面上方的快速標籤（如：讚美、救恩、受苦等）。
  2. 你也可以在搜尋框中輸入你想要的關鍵字。
  3. 系統會列出該主題常用的詩歌詞彙，並標示好粵拼與聲調，方便你直接套用進樂譜。

✏️ 0243填詞符號搜尋

採用填詞人黃志華的「0243」音高符號系統，按旋律音高查找詞語：

  1. 3 = 高音（粵語聲調 1、2）
  2. 0 = 中低音（聲調 4）
  3. 2 = 中音（聲調 3、5）
  4. 4 = 低音（聲調 6）
  5. 例：旋律是「中低→高」，就點擊 0 再點 3，找出如「耶穌」等詞語。
