# Canto-hymn-lyric-helper
Helper tool for writing Cantonese hymn lyrics, offering Jyutping-to-character lookup and tone-sequence matching.

✝ 粵語詩歌創作助手 (Cantonese Hymn Writer)

👉 **[點擊這裡立即使用線上版](https://chimerical-dodol-031671.netlify.app/)**

📖 關於此工具 (About)
這是一個由AI輔助研發，專為粵語詩歌創作者、敬拜主領與填詞人設計的網頁輔助工具。寫粵語歌最大的挑戰在於「啱音」（詞的聲調必須與旋律走向完全一致），本工具內建了豐富的基督教專屬詞庫，幫助你透過傳統聲調、0243 填詞符號或主題，快速找到適合填入旋律的字詞。

本專案將前端介面與詞庫資料庫徹底分離，確保輕量、快速，且在伺服器環境下完全於本地端瀏覽器運行，不收集任何使用者資料。

✨ 核心功能與使用教學 (Features & Tutorial)

本工具包含三個主要面板，點擊上方標籤即可切換：

1. 🎼 聲調查字 (Tone Search)

使用情境：你已經寫好了一段旋律，需要找對應粵語九聲六調的詞語填進去。

如何使用：

根據旋律的高低起伏，點擊畫面上的聲調數字按鈕（1至6）。介面上方附有「三碗半牛腩麵」的色彩記憶口訣。

例如：你想找一個先「低平」後「高平」的詞，就依序點擊 4 和 1。

點擊「查詞語」，系統會優先列出完全吻合的基督教詞彙（例如：「耶穌」、「全知」、「平安」）。

2. 🏷️ 主題詞庫 (Topical Dictionary)

使用情境：你缺乏靈感，需要特定主題的詞彙庫來激發寫詞方向。

如何使用：

直接點擊畫面上的快速標籤（如：讚美、敬拜、感恩、救恩等）。

系統會列出所有與該主題相關的常用詩歌詞彙與連接詞，並附上每個字的粵拼與聲調，方便你直接套用進樂譜中。

3. ✏️ 0243 填詞符號搜尋 (0243 System Search)

使用情境：習慣使用填詞人黃志華「0243」音高符號系統的創作者。

如何使用：

本工具將旋律音高分為四級：

3 = 高音（對應粵語聲調 1、2）

0 = 中低音（對應粵語聲調 4）

2 = 中音（對應粵語聲調 3、5）

4 = 低音（對應粵語聲調 6）

範例：若旋律為「中低音 → 高音」，點擊 0 再點 3，即可找到如「耶穌」(je4 sou1) 等詞語。系統會自動在背景將龐大的詞庫進行 0243 轉換與配對。

🛠️ 給開發者：如何在本地執行 (Local Setup)

這個專案採用極簡且專業的前後端分離架構：

技術棧：純 HTML5 + CSS3 + Vanilla JavaScript (無框架)。

資料庫：詞庫獨立儲存於 database.csv，由 JS 的 Fetch API 動態讀取並建立索引。

執行步驟

下載本專案的 .zip 檔，或使用 git clone 複製到本地。

⚠️ 重要：因瀏覽器安全性限制 (CORS)，請勿直接雙擊 index.html 開啟。

請使用編輯器（如 VS Code）的 Live Server 擴充套件，或部署至本地伺服器環境中運行，確保 database.csv 能被正確讀取。

🤝 貢獻 (Contributing)

這是一個由社群驅動的工具！如果你發現有漏掉的基督教常用詞彙、連接詞，或是想要擴充詞庫：

你完全不需要修改任何程式碼（index.html）。

只需打開 database.csv，按照 詞語,粵拼,聲調序列,分類,意思 的格式新增一行即可。

歡迎發起 Pull Request，將你整理的詞彙合併進來，造福更多創作者！
